# Code of Conduct

Leo is a small open-source project. To keep it welcoming for contributors,
everyone participating in issues, pull requests, and discussions is expected to:

- Be respectful and constructive.
- Assume good intent and ask before accusing.
- Focus critique on code and ideas, not people.
- Respect differing levels of experience.
- Accept maintainer decisions once a discussion has concluded, even when you
  disagree — escalate in a new issue if you think a call should be revisited.

Behavior that makes the project unsafe or unproductive for others is not
welcome. The maintainers reserve the right to remove comments, close issues,
or block contributors who repeatedly act in bad faith.

## Reporting

To report an incident or a concern, email `security@blackpaw.studio`.
Reports are handled confidentially by the maintainers.

## Scope

This applies to all project spaces — issues, pull requests, discussions,
code review comments, and any community channels that identify as part of
the Leo project.
